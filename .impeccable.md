## Design Context

### Users
- 짧은 시간에 로그인 후 스핀 결과를 빠르게 확인하는 캐주얼 웹 사용자
- 사용 맥락은 이동 중 모바일 또는 데스크톱에서 즉시 접속해 반복 플레이하는 상황
- 핵심 과업은 `스핀 결과 확인 -> 코인/레벨 관리 -> 다음 액션 선택`의 루프를 빠르게 수행하는 것

### Brand Personality
- 3단어 성격: `정돈됨`, `담백함`, `신뢰감`
- 톤앤매너: 자극적인 연출보다 명확한 정보 전달과 차분한 인터랙션 중심
- 목표 감정: "실물 기기처럼 안정적이고 깔끔하다"는 인상, 조작 시 불필요한 피로 없는 만족감

### Aesthetic Direction
- 일본 거리 자판기 레퍼런스 기반의 `화이트 바디 + 유리 진열창 + 규칙적인 상품 배열` 구조
- 전체 스타일은 미니멀 하드웨어 감성: 얇은 라인, 절제된 색, 낮은 채도, 균일한 간격
- 강한 네온/글로우/과장된 그라데이션보다, 실제 기기 UI처럼 기능 중심 계층을 우선
- 시각 포인트는 소수만 사용: 상태 강조(당첨/꽝), 주요 버튼, 핵심 결과 텍스트
- 인터페이스 언어는 기본적으로 한국어 유지(디자인 레퍼런스만 일본 자판기 무드 적용)

### Design Principles
1. 물성 우선: 화면은 "앱"보다 "실물 자판기 전면 패널"처럼 보여야 한다.
2. 절제된 강조: 강조 색과 굵은 요소는 핵심 상태/행동에만 제한적으로 사용한다.
3. 정렬의 일관성: 진열창, 상태 카드, 버튼은 동일한 그리드 리듬과 간격 체계를 유지한다.
4. 저자극 인터랙션: 모션은 기능 피드백 용도에 한정하고 이동 거리/속도를 최소화한다.
5. 읽기 쉬운 구조: 한 번에 현재 상태, 비용, 다음 행동을 이해할 수 있도록 정보 위계를 단순화한다.
